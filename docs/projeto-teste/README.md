# Visão Geral

## O que é

Descreva em 2-3 parágrafos o propósito do projeto, o problema que resolve e para quem é destinado.

## Quickstart

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/seu-projeto
cd seu-projeto

# Instale as dependências
npm install

# Rode localmente
npm start
```
